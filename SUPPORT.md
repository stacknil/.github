# Support

## Use the right channel

- Bug reports: use the bug report issue form
- Feature proposals: use the feature request form
- Documentation fixes: use the docs improvement form
- Security concerns: follow `SECURITY.md`
- Repository-specific usage questions: read the target repository README first

## Expectations

Support is provided on a best-effort basis.

Well-scoped, reproducible, and repository-relevant requests are easier to review and answer.
