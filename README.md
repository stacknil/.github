# stacknil/.github

Default community health files for repositories owned by **stacknil**.

This repository centralizes shared collaboration defaults, including:

- Code of Conduct
- Contributing Guidelines
- Security Policy
- Support Guidelines
- Issue Forms
- Pull Request Template

Repositories can override any default here by defining their own file locally.

Notes:

- This repository is meant to stay minimal and reusable.
- Content should remain general enough for software, security, systems, data, and research-adjacent repositories.
- Repository-specific policies should live in the target repository, not here.
- If a target repository defines its own community health file, that local file takes precedence over the default here.
- If a target repository defines its own `.github/ISSUE_TEMPLATE/` content, it should be treated as managing its own issue intake rather than inheriting the default issue form set from this repository.

Use local overrides when:

- A repository needs a repository-specific security or support path
- A repository is documentation-only, research-only, or otherwise not a general software support target
- A repository needs issue forms or pull request guidance that are materially different from the shared defaults
