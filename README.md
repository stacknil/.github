# stacknil/.github

Default community health files for repositories owned by **stacknil**.

This repository centralizes shared collaboration defaults, including:

- Code of Conduct
- Contributing Guidelines
- Security Policy
- Support Guidelines
- Issue Forms
- Pull Request Template

Repositories can override any default here by defining their own file locally.

Notes:

- This repository is meant to stay minimal and reusable.
- Content should remain general enough for software, security, systems, data, and research-adjacent repositories.
- Repository-specific policies should live in the target repository, not here.
